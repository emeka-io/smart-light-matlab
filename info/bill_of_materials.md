# Bill of Materials (BOM) For Smart Light Project

This list includes all the hardware components required to build and test the Smart Light System.

| Component | Quantity | Description/Purpose |
| :--- | :---: | :--- |
| **Microcontroller** | 1 | e.g., Arduino Uno or compatible board. Acts as the brain of the system. |
| **LDR (Photoresistor)** | 1 | Light-dependent resistor used to sense ambient light levels. |
| **LED** | 1 | Standard 5mm LED (Any color) to act as the smart light output. |
| **Push Button** | 1 | Tactile switch for the "Human Control" manual override. |
| **Resistor (10kΩ)** | 1 | Pull-down resistor for the LDR voltage divider circuit. |
| **Resistor (220Ω)** | 1 | Current-limiting resistor for the LED. |
| **Breadboard** | 1 | Full-size or Half-size for prototyping the circuit. |
| **Jumper Wires** | 1 set | Male-to-Male wires for making connections. |
| **USB Cable** | 1 | To connect the microcontroller to the PC for MATLAB communication. |

## Software Requirements
* **MATLAB** (R2021a or newer recommended)
* **MATLAB Support Package for Arduino Hardware**
* **Arduino IDE** (For initial firmware upload)
